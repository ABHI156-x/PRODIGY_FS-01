# PRODIGY_FS-01 – Full Stack Authentication System

This project is part of the Prodigy InfoTech Full Stack Development Internship (Task 01).

It is a full stack authentication system that supports user registration, login, email verification, and password reset.

## Tech Stack

Frontend:
- React (Vite)
- Context API
- Axios
- CSS

Backend:
- Node.js
- Express.js
- MongoDB
- JWT
- Bcrypt
- Nodemailer

## Features

- User Registration
- User Login & Logout
- Email Verification
- Password Reset via Email
- JWT Authentication
- Protected Routes

## Project Structure

client/ – Frontend  
server/ – Backend
